# CS234 - 强化学习 公式整理

## 1. 马尔可夫决策过程 (MDP)

### 1.1 MDP 定义

MDP 由五元组 $(S, A, P, R, \gamma)$ 定义：

- $S$：状态空间（State space）
- $A$：动作空间（Action space）
- $P$：状态转移概率 $P(s' | s, a)$
- $R$：奖励函数 $R(s, a, s')$
- $\gamma \in [0, 1)$：折扣因子（Discount factor）

### 1.2 策略 (Policy)

- **确定性策略**：$\pi: S \to A$，给定状态 $s$，直接输出一个确定的动作 $a$。
- **随机策略**：$\pi(a|s) = P(A_t = a | S_t = s)$，给定状态 $s$，输出每个动作的选择概率。

### 1.3 回报 (Return)

$$G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \cdots = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$$

从时刻 $t$ 开始，未来所有奖励的折扣累加和。$\gamma$ 越小，智能体越"短视"，更看重近期奖励；$\gamma$ 越接近 1，越看重长期收益。

## 2. 价值函数 (Value Functions)

### 2.1 状态价值函数 (State Value Function)

$$V^{\pi}(s) = \mathbb{E}_{\pi}\left[G_t \mid S_t = s\right] = \mathbb{E}_{\pi}\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \mid S_t = s\right]$$

在策略 $\pi$ 下，从状态 $s$ 出发，未来能获得的期望累计回报。衡量的是"处于某个状态有多好"。 动作价值函数 (Action Value Function / Q-Function)

$$Q^{\pi}(s, a) = \mathbb{E}_{\pi}\left[G_t \mid S_t = s, A_t = a\right] = \mathbb{E}_{\pi}\left[\sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \mid S_t = s, A_t = a\right]$$

在策略 $\pi$ 下，从状态 $s$ 执行动作 $a$ 后，未来能获得的期望累计回报。衡量的是"在某个状态下采取某个动作有多好"。 V 和 Q 的关系

$$V^{\pi}(s) = \sum_{a \in A} \pi(a|s) \, Q^{\pi}(s, a)$$

状态价值 = 所有动作的 Q 值按策略概率加权平均。即"这个状态有多好"等于"各个动作有多好"的加权平均。

$$Q^{\pi}(s, a) = R(s, a) + \gamma \sum_{s' \in S} P(s'|s, a) \, V^{\pi}(s')$$

动作价值 = 即时奖励 + 折扣后的下一状态价值的期望。执行动作 $a$ 后，获得即时奖励 $R(s,a)$，然后以概率 $P(s'|s,a)$ 转移到各个后继状态 $s'$。

## 3. 贝尔曼方程 (Bellman Equations)

### 3.1 贝尔曼期望方程 (Bellman Expectation Equation)

**状态价值形式**：

$$V^{\pi}(s) = \sum_{a \in A} \pi(a|s) \left[ R(s, a) + \gamma \sum_{s' \in S} P(s'|s, a) \, V^{\pi}(s') \right]$$

将 V 和 Q 的关系代入，得到 V 关于自身的递归方程：当前状态的价值可以用后继状态的价值来表示。这是动态规划求解的基础。

**动作价值形式**：

$$Q^{\pi}(s, a) = R(s, a) + \gamma \sum_{s' \in S} P(s'|s, a) \sum_{a' \in A} \pi(a'|s') \, Q^{\pi}(s', a')$$

类似地，Q 值也可以用后继状态的 Q 值递归表示：执行动作 $a$ 获得奖励后，转移到 $s'$，在 $s'$ 按策略 $\pi$ 选择 $a'$ 继续。

### 3.2 贝尔曼最优方程 (Bellman Optimality Equation)

**最优状态价值**：

$$V^*(s) = \max_{a \in A} \left[ R(s, a) + \gamma \sum_{s' \in S} P(s'|s, a) \, V^*(s') \right]$$

与贝尔曼期望方程不同，这里用 $\max$ 替代了对策略的求和——不再按概率加权平均，而是直接选最好的动作。

**最优动作价值**：

$$Q^*(s, a) = R(s, a) + \gamma \sum_{s' \in S} P(s'|s, a) \max_{a' \in A} Q^*(s', a')$$

在后继状态 $s'$ 也选最优动作（$\max$），而非按策略概率求期望。

**最优策略**：

$$\pi^*(s) = \arg\max_{a \in A} Q^*(s, a)$$

有了最优 Q 值后，最优策略就是每个状态下贪心地选 Q 值最大的动作。

## 4. 动态规划 (Dynamic Programming)

### 4.1 策略评估 (Policy Evaluation)

迭代更新：

$$V_{k+1}(s) = \sum_{a \in A} \pi(a|s) \left[ R(s, a) + \gamma \sum_{s' \in S} P(s'|s, a) \, V_k(s') \right]$$

给定固定策略 $\pi$，反复用贝尔曼期望方程更新每个状态的值，直到 $V$ 收敛。每一轮用上一轮的 $V_k$ 计算出新的 $V_{k+1}$。 策略改进 (Policy Improvement)

$$\pi'(s) = \arg\max_{a \in A} Q^{\pi}(s, a) = \arg\max_{a \in A} \left[ R(s, a) + \gamma \sum_{s' \in S} P(s'|s, a) \, V^{\pi}(s') \right]$$

对当前策略 $\pi$ 的价值函数，在每个状态贪心地选 Q 值最大的动作，构成新策略 $\pi'$。注意 $\pi'(s)$ 是确定性策略，返回的是一个具体动作。

**策略改进定理**：若 $Q^{\pi}(s, \pi'(s)) \geq V^{\pi}(s)$ 对所有 $s$ 成立，则 $V^{\pi'}(s) \geq V^{\pi}(s)$。

即：如果在每个状态，按新策略选的动作的 Q 值都不低于当前状态价值（最大值 $\geq$ 加权平均值，自然成立），那么新策略一定不比旧策略差。

### 4.3 值迭代 (Value Iteration)

$$V_{k+1}(s) = \max_{a \in A} \left[ R(s, a) + \gamma \sum_{s' \in S} P(s'|s, a) \, V_k(s') \right]$$

每一步直接选最优动作更新值函数，将策略评估和策略改进合并为一步。不需要显式维护策略，收敛后从 $V^*$ 即可提取最优策略。

收敛条件：$\|V_{k+1} - V_k\|_{\infty} < \theta$（$\theta$ 为阈值）

即所有状态上值函数的最大变化量小于阈值 $\theta$ 时停止迭代。

**收缩映射定理**：贝尔曼最优算子 $T$ 是 $\gamma$-收缩映射：

$$\|T V_1 - T V_2\|_{\infty} \leq \gamma \|V_1 - V_2\|_{\infty}$$

对任意两个值函数 $V_1, V_2$，经过算子 $T$ 作用后它们之间的差距缩小为原来的 $\gamma$ 倍。因为 $\gamma < 1$，所以反复迭代一定收敛到唯一不动点 $V^*$。

## 5. 无模型预测 (Model-Free Prediction)

### 5.1 蒙特卡洛方法 (Monte Carlo)

**首次访问 MC**（First-Visit MC）：

$$V(s) \leftarrow V(s) + \alpha \left[ G_t - V(s) \right]$$

其中 $G_t$ 是从状态 $s$ 开始的实际回报。每轮完整 episode 结束后，用实际回报 $G_t$ 与当前估计值 $V(s)$ 的差来更新。$\alpha$ 是学习率，控制每次更新的步幅。

**增量均值更新**：

$$V(S_t) \leftarrow V(S_t) + \frac{1}{N(S_t)} \left[ G_t - V(S_t) \right]$$

当学习率取为 $1/N(S_t)$（$N$ 为访问次数）时，等价于对所有历史回报求算术平均。

### 5.2 时序差分学习 (Temporal Difference, TD)

**TD(0)**：

$$V(S_t) \leftarrow V(S_t) + \alpha \left[ R_{t+1} + \gamma V(S_{t+1}) - V(S_t) \right]$$

与 MC 不同，TD 不需要等 episode 结束，每走一步就用"即时奖励 + 下一状态估计值"来更新当前状态的值。这就是"自举"（bootstrapping）——用估计值更新估计值。

- **TD 目标**：$R_{t+1} + \gamma V(S_{t+1})$，对真实回报 $G_t$ 的一步近似估计。
- **TD 误差**：$\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$，预测的"惊喜"程度，正值表示实际比预期好。

### 5.3 n-步 TD (n-step TD)

$$G_t^{(n)} = R_{t+1} + \gamma R_{t+2} + \cdots + \gamma^{n-1} R_{t+n} + \gamma^n V(S_{t+n})$$

$$V(S_t) \leftarrow V(S_t) + \alpha \left[ G_t^{(n)} - V(S_t) \right]$$

TD(0) 只看一步，MC 看到底。n-步 TD 是两者的折中：向前实际走 $n$ 步收集真实奖励，之后用 $V(S_{t+n})$ 自举。$n$ 越大越像 MC，$n=1$ 就是 TD(0)。 TD(λ)

**λ-回报**：

$$G_t^{\lambda} = (1 - \lambda) \sum_{n=1}^{\infty} \lambda^{n-1} G_t^{(n)}$$

对所有 $n$-步回报做加权平均，权重按 $\lambda^{n-1}$ 指数衰减，$(1-\lambda)$ 是归一化系数。$\lambda=0$ 退化为 TD(0)，$\lambda=1$ 退化为 MC。

**资格迹 (Eligibility Traces)**：

$$e_0(s) = 0$$

$$e_t(s) = \gamma \lambda \, e_{t-1}(s) + \mathbf{1}(S_t = s)$$

资格迹记录每个状态的"近期访问频率"：每次访问状态 $s$ 时迹值加 1，之后以 $\gamma\lambda$ 的速率衰减。迹值越高，说明该状态最近被频繁访问，更新幅度越大。

**后向视角更新**：

$$\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$$

$$V(s) \leftarrow V(s) + \alpha \, \delta_t \, e_t(s), \quad \forall s$$

每一步的 TD 误差 $\delta_t$ 会按资格迹的比例分摊给所有状态：最近访问过且频繁出现的状态获得更多更新。这是 $\lambda$-回报前向视角的等价高效实现。

## 6. 无模型控制 (Model-Free Control)

### 6.1 SARSA (On-Policy TD Control)

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma Q(S_{t+1}, A_{t+1}) - Q(S_t, A_t) \right]$$

在状态 $S_t$ 执行动作 $A_t$，观察奖励 $R_{t+1}$ 和下一状态 $S_{t+1}$，再按策略选出下一动作 $A_{t+1}$，用 $(S_t, A_t, R_{t+1}, S_{t+1}, A_{t+1})$ 这五元组更新 Q 值——这就是 SARSA 名字的由来。

使用 $\epsilon$-贪心策略选择动作。由于行为策略和学习策略是同一个（$\epsilon$-贪心），所以是 on-policy 方法。

### 6.2 Q-Learning (Off-Policy TD Control)

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma \max_{a'} Q(S_{t+1}, a') - Q(S_t, A_t) \right]$$

与 SARSA 的关键区别：更新目标中用 $\max_{a'} Q$ 而非实际选的下一动作。即使行为策略带有探索（$\epsilon$-贪心），学习目标始终对准最优策略（贪心），因此是 off-policy 方法。

- 行为策略：$\epsilon$-贪心（用于采集数据，保证探索）
- 目标策略：贪心 $\pi(s) = \arg\max_a Q(s, a)$（要学习的最优策略）

### 6.3 Expected SARSA

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha \left[ R_{t+1} + \gamma \sum_{a'} \pi(a'|S_{t+1}) Q(S_{t+1}, a') - Q(S_t, A_t) \right]$$

SARSA 用单个下一动作的 Q 值，Q-Learning 用最大 Q 值，Expected SARSA 则用所有动作 Q 值的期望（按策略概率加权）。方差比 SARSA 低，是 SARSA 和 Q-Learning 之间的折中。 ε-贪心策略 (ε-Greedy)

$$\pi(a|s) = \begin{cases} 1 - \epsilon + \frac{\epsilon}{|A|} & \text{if } a = \arg\max_{a'} Q(s, a') \\ \frac{\epsilon}{|A|} & \text{otherwise} \end{cases}$$

以 $1-\epsilon$ 的概率选当前最优动作（利用），以 $\epsilon$ 的概率在所有动作中均匀随机选（探索）。$\epsilon$ 控制探索程度：$\epsilon=0$ 是纯贪心，$\epsilon=1$ 是纯随机。实践中常让 $\epsilon$ 随训练逐渐衰减。

## 7. 价值函数近似 (Value Function Approximation)

### 7.1 线性近似

$$\hat{V}(s; \mathbf{w}) = \mathbf{w}^T \mathbf{x}(s) = \sum_{j=1}^{d} w_j x_j(s)$$

$$\hat{Q}(s, a; \mathbf{w}) = \mathbf{w}^T \mathbf{x}(s, a)$$

用参数 $\mathbf{w}$ 和特征向量 $\mathbf{x}(s)$ 的线性组合来近似值函数。当状态空间太大（如连续状态）无法为每个状态存一个值时，用函数近似来泛化。 梯度下降更新

**目标**：最小化均方误差

$$J(\mathbf{w}) = \mathbb{E}_{\pi}\left[\left(V^{\pi}(s) - \hat{V}(s; \mathbf{w})\right)^2\right]$$

衡量近似值函数 $\hat{V}$ 与真实值函数 $V^\pi$ 之间的差距。

**随机梯度下降(SGD)**：

$$\mathbf{w} \leftarrow \mathbf{w} + \alpha \left[ V^{\pi}(s) - \hat{V}(s; \mathbf{w}) \right] \nabla_{\mathbf{w}} \hat{V}(s; \mathbf{w})$$

沿着减小误差的方向调整参数。但真实 $V^\pi(s)$ 未知，需要用样本来替代。

**MC + 函数近似**：

$$\mathbf{w} \leftarrow \mathbf{w} + \alpha \left[ G_t - \hat{V}(S_t; \mathbf{w}) \right] \nabla_{\mathbf{w}} \hat{V}(S_t; \mathbf{w})$$

用实际回报 $G_t$ 作为 $V^\pi$ 的无偏估计来替代。收敛到局部最优，但方差大、需要完整 episode。

**TD(0) + 函数近似（半梯度）**：

$$\mathbf{w} \leftarrow \mathbf{w} + \alpha \left[ R_{t+1} + \gamma \hat{V}(S_{t+1}; \mathbf{w}) - \hat{V}(S_t; \mathbf{w}) \right] \nabla_{\mathbf{w}} \hat{V}(S_t; \mathbf{w})$$

用 TD 目标替代 $V^\pi$。称为"半梯度"是因为只对 $\hat{V}(S_t; \mathbf{w})$ 求梯度，忽略了目标中 $\hat{V}(S_{t+1}; \mathbf{w})$ 对 $\mathbf{w}$ 的依赖。方差更低，可在线学习。

### 7.3 DQN (Deep Q-Network)

**损失函数**：

$$L(\mathbf{w}) = \mathbb{E}\left[\left(R_{t+1} + \gamma \max_{a'} Q(S_{t+1}, a'; \mathbf{w}^{-}) - Q(S_t, A_t; \mathbf{w})\right)^2\right]$$

用深度神经网络近似 Q 函数。两个关键技巧保证训练稳定性：

- $\mathbf{w}^{-}$：目标网络参数（定期从 $\mathbf{w}$ 复制），计算 TD 目标时用固定的旧参数，避免"自己追自己"导致的不稳定。
- **经验回放 (Experience Replay)**：从 buffer $\mathcal{D}$ 中均匀采样 $(s, a, r, s')$，打破样本间的时间相关性，让训练更稳定。

## 8. 策略梯度 (Policy Gradient)

### 8.1 策略参数化

$$\pi_{\theta}(a|s) = \frac{e^{\theta^T \mathbf{x}(s,a)}}{\sum_{a'} e^{\theta^T \mathbf{x}(s,a')}} \quad \text{(Softmax 策略)}$$

将每个动作的偏好值 $\theta^T \mathbf{x}(s,a)$ 通过 softmax 转为概率。偏好值越高的动作被选择的概率越大，且保证概率和为 1。适用于离散动作空间。

$$\pi_{\theta}(a|s) = \mathcal{N}(\mu_{\theta}(s), \sigma^2) \quad \text{(高斯策略，连续动作)}$$

动作从均值为 $\mu_\theta(s)$、方差为 $\sigma^2$ 的正态分布中采样。适用于连续动作空间（如机器人控制中的力矩、速度等）。

### 8.2 策略目标函数

$$J(\theta) = \mathbb{E}_{\pi_{\theta}}\left[\sum_{t=0}^{\infty} \gamma^t R_{t+1}\right] = \sum_{s} d^{\pi}(s) V^{\pi}(s)$$

其中 $d^{\pi}(s)$ 是策略 $\pi$ 下的稳态分布（智能体长期运行后访问各状态的频率）。目标是直接优化策略参数 $\theta$ 使得期望累计回报最大。

### 8.3 策略梯度定理 (Policy Gradient Theorem)

$$\nabla_{\theta} J(\theta) = \mathbb{E}_{\pi_{\theta}}\left[\sum_{t=0}^{\infty} \gamma^t \, Q^{\pi_{\theta}}(S_t, A_t) \, \nabla_{\theta} \ln \pi_{\theta}(A_t | S_t)\right]$$

简化形式：

$$\nabla_{\theta} J(\theta) = \mathbb{E}_{\pi_{\theta}}\left[Q^{\pi_{\theta}}(s, a) \, \nabla_{\theta} \ln \pi_{\theta}(a | s)\right]$$

核心思想：梯度方向由两部分决定——$\nabla_\theta \ln \pi_\theta(a|s)$ 指示如何调整参数使动作 $a$ 的概率增大，$Q^{\pi}(s,a)$ 决定增大多少。Q 值高的动作概率被增大，Q 值低的被减小。不需要知道环境模型。

### 8.4 REINFORCE 算法

$$\theta \leftarrow \theta + \alpha \, \gamma^t \, G_t \, \nabla_{\theta} \ln \pi_{\theta}(A_t | S_t)$$

用实际回报 $G_t$ 替代策略梯度定理中的 $Q^\pi$。每次完整 episode 结束后，用采样得到的回报更新参数。简单但方差大。

**带基线的 REINFORCE**：

$$\theta \leftarrow \theta + \alpha \, \gamma^t \, (G_t - b(S_t)) \, \nabla_{\theta} \ln \pi_{\theta}(A_t | S_t)$$

常用基线 $b(s) = V(s)$，此时 $G_t - b(S_t)$ 近似**优势函数** $A^{\pi}(s,a)$。

减去基线不改变梯度期望（无偏），但能显著降低方差：回报高于平均水平的动作被增强，低于平均的被削弱。

### 8.5 优势函数 (Advantage Function)

$$A^{\pi}(s, a) = Q^{\pi}(s, a) - V^{\pi}(s)$$

衡量动作 $a$ 相对于平均水平的好坏：正值表示该动作优于平均，负值表示劣于平均。比直接用 $Q$ 值更稳定，因为消除了状态本身价值的影响。

性质：$\sum_a \pi(a|s) A^{\pi}(s,a) = 0$，即所有动作的优势按策略概率加权平均为零。

## 9. Actor-Critic 方法

### 9.1 Actor-Critic 更新

**Critic 更新（TD(0)）**：

$$\delta_t = R_{t+1} + \gamma \hat{V}(S_{t+1}; \mathbf{w}) - \hat{V}(S_t; \mathbf{w})$$

$$\mathbf{w} \leftarrow \mathbf{w} + \alpha_w \, \delta_t \, \nabla_{\mathbf{w}} \hat{V}(S_t; \mathbf{w})$$

Critic（评论家）用 TD 方法学习值函数，作为对当前策略好坏的评估。$\delta_t$ 就是 TD 误差。

**Actor 更新**：

$$\theta \leftarrow \theta + \alpha_{\theta} \, \gamma^t \, \delta_t \, \nabla_{\theta} \ln \pi_{\theta}(A_t | S_t)$$

Actor（演员）用 Critic 提供的 TD 误差 $\delta_t$ 作为优势估计来更新策略。TD 误差为正说明这个动作比预期好，增大其概率；为负则减小。相比 REINFORCE，不需要等 episode 结束，可以每步在线更新。

### 9.2 A2C / A3C

**优势 Actor-Critic**：用 $\hat{A}(S_t, A_t) = \delta_t$ 作为优势估计

$$\theta \leftarrow \theta + \alpha \, \hat{A}(S_t, A_t) \, \nabla_{\theta} \ln \pi_{\theta}(A_t | S_t)$$

A2C 是同步版本，多个 worker 并行采集数据后汇总更新；A3C 是异步版本，各 worker 独立更新共享参数。通过并行化提高数据采集效率和训练稳定性。

## 10. 探索与利用 (Exploration vs Exploitation)

### 10.1 多臂老虎机 (Multi-Armed Bandit)

**动作价值估计**：

$$Q_t(a) = \frac{\sum_{i=1}^{t-1} R_i \cdot \mathbf{1}(A_i = a)}{\sum_{i=1}^{t-1} \mathbf{1}(A_i = a)}$$

用过去选择动作 $a$ 获得的奖励的算术平均来估计其价值。分子是历史总奖励，分母是选择次数。 UCB (Upper Confidence Bound)

$$A_t = \arg\max_a \left[ Q_t(a) + c \sqrt{\frac{\ln t}{N_t(a)}} \right]$$

选择"估计值 + 不确定性上界"最大的动作。第二项是探索加成：$N_t(a)$ 越小（尝试越少），不确定性越大，越倾向探索。$c$ 控制探索力度。随着 $N_t(a)$ 增大，探索项衰减，逐渐转向利用。

- $N_t(a)$：动作 $a$ 被选择的次数
- $c$：探索系数

### 10.3 Hoeffding 不等式

$$P\left[Q(a) > Q_t(a) + U_t(a)\right] \leq e^{-2 N_t(a) U_t(a)^2}$$

UCB 的理论基础：真实值 $Q(a)$ 超过估计值加上界 $U_t(a)$ 的概率随采样次数指数衰减。采样越多，估计越准，上界越紧。 Thompson Sampling

对每个动作 $a$，维护后验分布 $P(Q(a) | \text{data})$，每步从后验采样选择：

$$A_t = \arg\max_a \hat{Q}_a, \quad \hat{Q}_a \sim P(Q(a) | \text{data})$$

一种贝叶斯探索方法：对每个动作维护一个关于其真实值的后验分布，每步从后验采样一个值，选采样值最大的动作。不确定性大的动作，采样值方差大，有时会被选到（探索）；随着数据增多，后验变窄，自然转向利用。 遗憾 (Regret)

$$\text{Regret}_T = T \cdot V^* - \sum_{t=1}^{T} \mathbb{E}[R_t]$$

其中 $V^* = \max_a Q(a)$。

累计遗憾 = 如果每步都选最优动作能获得的总奖励 $-$ 实际获得的总期望奖励。遗憾越低说明算法越好。好的探索算法（如 UCB、Thompson Sampling）能实现遗憾以 $O(\ln T)$ 的速度增长，而纯随机策略则是 $O(T)$。

## 11. 重要性采样 (Importance Sampling)

### 11.1 重要性采样比率

$$\rho_{t:T-1} = \prod_{k=t}^{T-1} \frac{\pi(A_k | S_k)}{b(A_k | S_k)}$$

- $\pi$：目标策略
- $b$：行为策略

### 11.2 普通重要性采样 (Ordinary IS)

$$V(s) = \frac{\sum_{t \in \mathcal{T}(s)} \rho_{t:T(t)-1} \, G_t}{|\mathcal{T}(s)|}$$

### 11.3 加权重要性采样 (Weighted IS)

$$V(s) = \frac{\sum_{t \in \mathcal{T}(s)} \rho_{t:T(t)-1} \, G_t}{\sum_{t \in \mathcal{T}(s)} \rho_{t:T(t)-1}}$$

加权 IS 偏差更小，方差更低，实际中更常用。

## 12. 批量强化学习与模仿学习

### 12.1 Fitted Q-Iteration

给定数据集 $\mathcal{D} = \{(s_i, a_i, r_i, s_i')\}$，迭代：

$$Q_{k+1} \leftarrow \arg\min_Q \sum_i \left( Q(s_i, a_i) - \left[ r_i + \gamma \max_{a'} Q_k(s_i', a') \right] \right)^2$$

### 12.2 行为克隆 (Behavioral Cloning)

$$\theta^* = \arg\max_{\theta} \sum_{(s, a) \in \mathcal{D}} \ln \pi_{\theta}(a | s)$$
